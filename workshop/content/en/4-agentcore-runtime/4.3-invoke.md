## Invoke one trader

```powershell
agentcore invoke "Warren"
```

Expected result:

```json
{
  "status": "completed",
  "traders": [
    {"trader": "Warren", "result": "..."}
  ]
}
```

Run a complete cycle:

```powershell
agentcore invoke "all"
```

### EventBridge Scheduler

For the first iteration, create the schedule in the AWS Console to avoid target ARN syntax differences between versions:

1. Open **Amazon EventBridge → Scheduler → Create schedule**.
2. Choose `rate(60 minutes)`.
3. Select **Universal target** and the AgentCore Runtime invoke operation.
4. Use the Runtime ARN returned by `agentcore status`.
5. Set the input to `{}`.
6. Create an execution role that can invoke only this Runtime.

{{% notice warning %}}
Use a long interval during the workshop and disable the schedule when it is not needed to avoid unintended model invocations.
{{% /notice %}}
