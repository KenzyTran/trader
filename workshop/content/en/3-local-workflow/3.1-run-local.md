## Local configuration

Select the current model provider in `.env`:

```dotenv
MODEL_PROVIDER=openai
MODEL_ID=gpt-4.1-mini
OPENAI_API_KEY=...
RUN_EVEN_WHEN_MARKET_IS_CLOSED=true
```

If `MASSIVE_API_KEY` is not configured, the project automatically uses the market simulator. Run one cycle:

```powershell
uv run trader once
```

Then start the API and dashboard in two terminals:

```powershell
# Terminal 1
uv run trader api

# Terminal 2
cd frontend
npm ci
npm run dev
```

Open `http://localhost:5173` and verify that:

- Four traders are displayed.
- Portfolio value and cash load from the API.
- The activity log records `Run started` and `Run ended`.
- If Telegram is configured, the chat receives a summary.

{{% notice tip %}}
Always validate the workflow locally before deployment. AgentCore does not fix existing prompt, tool, or model-configuration problems in the application.
{{% /notice %}}
