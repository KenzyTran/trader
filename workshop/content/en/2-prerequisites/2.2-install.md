## Install the source code

Clone the repository with the Hugo theme submodule:

```powershell
git clone --recurse-submodules <YOUR_REPOSITORY_URL>
cd trader
uv sync --extra dev
Copy-Item .env.example .env
```

Install the AgentCore SDK and CLI:

```powershell
uv pip install bedrock-agentcore
npm install -g @aws/agentcore
```

Verify the project before deployment:

```powershell
uv run pytest
uv run ruff check src tests
uv run trader reset
```

Expected result: the tests pass and four accounts are initialized in `data/accounts.db`.
