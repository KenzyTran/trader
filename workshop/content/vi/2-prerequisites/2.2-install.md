## Cài đặt source code

Clone cả Hugo theme submodule:

```powershell
git clone --recurse-submodules <YOUR_REPOSITORY_URL>
cd trader
uv sync --extra dev
Copy-Item .env.example .env
```

Cài AgentCore SDK và CLI:

```powershell
uv pip install bedrock-agentcore
npm install -g @aws/agentcore
```

Kiểm tra project trước khi deploy:

```powershell
uv run pytest
uv run ruff check src tests
uv run trader reset
```

Kết quả mong đợi: test pass và bốn tài khoản được khởi tạo trong `data/accounts.db`.
