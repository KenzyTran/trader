## Yêu cầu trước khi bắt đầu

### Tài khoản và quyền

- Một AWS account không sử dụng root user cho thao tác hằng ngày.
- AWS credentials đã cấu hình bằng IAM Identity Center hoặc profile riêng.
- Quyền tạo AgentCore Runtime, IAM role, CodeBuild, CloudWatch và DynamoDB.
- Model access cho model được đặt trong `MODEL_ID` tại Region đã chọn.

### Công cụ

| Công cụ | Phiên bản tối thiểu | Kiểm tra |
|---|---:|---|
| Python | 3.12 | `python --version` |
| uv | mới nhất | `uv --version` |
| AWS CLI | v2 | `aws --version` |
| Node.js | 20 | `node --version` |
| AgentCore CLI | mới nhất | `agentcore --version` |
| Hugo Extended | 0.139+ | `hugo version` |

### Region

Toàn bộ tài nguyên và model phải nằm cùng Region. Workshop dùng `us-west-2` trong ví dụ:

```powershell
aws configure set region us-west-2 --profile trader-workshop
$env:AWS_PROFILE="trader-workshop"
aws sts get-caller-identity
```

{{% notice warning %}}
Không đưa access key, API key hoặc Telegram bot token vào Git. Chỉ lưu chúng trong `.env` local hoặc
credential provider/Secrets Manager khi chạy cloud.
{{% /notice %}}
