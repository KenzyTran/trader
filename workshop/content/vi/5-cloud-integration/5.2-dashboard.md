## Phân phối dashboard

Frontend trong `frontend/` là static TypeScript/Vite application. Build production:

```powershell
cd frontend
npm ci
npm run build
```

Upload `dist/` lên một S3 bucket private và đặt CloudFront phía trước. Không bật public website hosting
trực tiếp trên bucket.

### API read-only

Dashboard chỉ cần các endpoint:

```text
GET /api/market
GET /api/traders
GET /api/traders/{name}
GET /api/traders/{name}/logs
```

Trong target architecture, API Gateway gọi Lambda adapter đọc DynamoDB. Không cho dashboard gọi tool
mua/bán hoặc invoke Runtime trực tiếp.

### Authentication

1. Tạo Cognito User Pool và App Client không có client secret cho browser.
2. Cấu hình API Gateway JWT authorizer.
3. Cho CloudFront route `/api/*` đến API Gateway và các path còn lại đến S3.
4. Chỉ cho phép `GET` trên dashboard API.

{{% notice info %}}
Nếu workshop chỉ tập trung AgentCore Runtime, bạn có thể tiếp tục chạy dashboard local và trỏ API local
vào DynamoDB. CloudFront/Cognito là bước mở rộng, không phải điều kiện để hoàn thành Runtime lab.
{{% /notice %}}
