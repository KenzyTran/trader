## AgentCore entrypoint

Entrypoint nằm tại `deploy/agentcore/runtime.py` và dùng `BedrockAgentCoreApp`:

```python
app = BedrockAgentCoreApp()

@app.entrypoint
async def invoke(payload):
    trader_name = payload.get("trader")
    return {"status": "completed", "traders": await run_traders(trader_name)}
```

Một payload không có `trader` chạy cả bốn agent; payload có tên chỉ chạy một trader.

### Tạo project AgentCore

AgentCore CLI quản lý deployment bằng một project riêng. Từ thư mục gốc repository:

```powershell
cd deploy/agentcore
agentcore create `
  --name TraderFloor `
  --framework Strands `
  --protocol HTTP `
  --model-provider Bedrock `
  --memory none `
  --build CodeZip
```

Lệnh tạo thư mục `TraderFloor` với cấu hình CDK và agent mẫu. Đồng bộ source trader cùng entrypoint của
workshop vào project vừa tạo:

```powershell
.\prepare_project.ps1
cd TraderFloor
```

Mở `agentcore/.env.local` và thêm các biến trong `../.env.example`. Sau đó validate và chạy local:

```powershell
agentcore validate
agentcore dev --logs
```

Trong terminal khác:

```powershell
agentcore dev "Warren"
```

### Deploy lên AWS

```powershell
agentcore deploy --plan
agentcore deploy
agentcore status
```

AgentCore CLI đóng gói CodeZip, synthesize CDK/CloudFormation, tạo IAM role và AgentCore Runtime rồi trả
về Runtime ARN.

{{% notice info %}}
Không cần Docker cho CodeZip. Chỉ chọn Container khi agent cần system dependency hoặc image riêng.
{{% /notice %}}
