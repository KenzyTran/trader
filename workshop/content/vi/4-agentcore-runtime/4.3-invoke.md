## Invoke một trader

```powershell
agentcore invoke "Warren"
```

Kết quả mong đợi:

```json
{
  "status": "completed",
  "traders": [
    {"trader": "Warren", "result": "..."}
  ]
}
```

Chạy đủ một cycle:

```powershell
agentcore invoke "all"
```

### EventBridge Scheduler

Trong bước đầu, tạo schedule bằng AWS Console để tránh phụ thuộc cú pháp ARN target giữa các phiên bản:

1. Mở **Amazon EventBridge → Scheduler → Create schedule**.
2. Chọn lịch `rate(60 minutes)`.
3. Chọn **Universal target** và operation invoke của AgentCore Runtime.
4. Dùng Runtime ARN trả về từ `agentcore status`.
5. Input là `{}`.
6. Tạo execution role chỉ có quyền invoke Runtime này.

{{% notice warning %}}
Đặt lịch dài trong workshop và tắt schedule khi không sử dụng để tránh gọi model ngoài ý muốn.
{{% /notice %}}
