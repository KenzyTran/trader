## Cấu hình local

Trong `.env`, chọn model provider hiện tại:

```dotenv
MODEL_PROVIDER=openai
MODEL_ID=gpt-4.1-mini
OPENAI_API_KEY=...
RUN_EVEN_WHEN_MARKET_IS_CLOSED=true
```

Nếu chưa có `MASSIVE_API_KEY`, project tự dùng market simulator. Chạy một cycle:

```powershell
uv run trader once
```

Sau đó mở API và dashboard ở hai terminal:

```powershell
# Terminal 1
uv run trader api

# Terminal 2
cd frontend
npm ci
npm run dev
```

Truy cập `http://localhost:5173` và xác minh:

- Bốn trader xuất hiện.
- Portfolio value và cash tải được từ API.
- Activity log ghi nhận `Run started` và `Run ended`.
- Nếu Telegram đã cấu hình, chat nhận được bản tóm tắt.

{{% notice tip %}}
Luôn xác minh workflow local trước khi deploy. AgentCore không sửa lỗi prompt, tool hoặc cấu hình model
có sẵn trong ứng dụng.
{{% /notice %}}
